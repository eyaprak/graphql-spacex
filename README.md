# SpaceX App

## General info
This project has made with `React.js`, `Node.js`, `Express.js`, `GraphQL` technologies.
To see GraphQL server `localhost:5000/graphql`.

## Setup
To run this project, install it locally using npm:
```
$ cd ../graphql-spacex
$ cd ./server
$ npm i
$ cd ./client
$ npm i
$ cd ./server
$ npm run dev
```
